Place files from Google Drive in this folder and reference them as /media/<filename> in Next.js.

Example usage:
- image: src="/media/team-photo.jpg"
- video: src="/media/highlights.mp4"

Recommended naming:
- hero-video.mp4
- hero-poster.jpg
- mission-bg.jpg
- service-athlete.jpg
- service-event.jpg
- service-marketing.jpg
